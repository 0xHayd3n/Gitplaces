Key design decisions
  • 8 domain groups vs 24 ecosystem groups — Domain collapses the long tail into Specialty
  • Preference persisted — saved to localStorage so it survives app restarts
  • Data model — each language gets both domainCategory and ecosystemCategory fields; current category field becomes ecosystemCategory
  • Default mode — Domain (simpler for new users), Ecosystem for power users
